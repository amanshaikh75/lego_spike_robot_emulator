# Vue 3 + Vite

This template should help get you started developing with Vue 3 in Vite.
The template uses Vue 3 `<script setup>` SFCs,
check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup)
to learn more.

Learn more about IDE Support for Vue in the
[Vue Docs Scaling up Guide](https://vuejs.org/guide/scaling-up/tooling.html#ide-support).

## Starting the server

1. Install node.js from https://nodejs.org/en/download if node.js does not exist on the machine.
2. Open Window Power Shell.
3. Run the following commands
```bash
cd C:\Users\nafis\github_repos\lego_spike_robot_simulator
npm install
npm dev run
```

Access the server in a browser by visiting 'http://localhost:5173'.
